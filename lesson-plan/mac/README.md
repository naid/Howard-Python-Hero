# Mac learning path (macOS)

Complete these three modules **in order** before starting shared lessons in [`../lessons/`](../lessons/).

| Order | Module | Goal |
|-------|--------|------|
| 1 | [01-system-setup](01-system-setup.md) | Terminal, Xcode CLI tools, Homebrew |
| 2 | [02-python-environment](02-python-environment.md) | Python 3.11+, venv, pip, first script |
| 3 | [03-gpu-and-accelerators](03-gpu-and-accelerators.md) | PyTorch on Mac (MPS for Apple Silicon) |

## Who this track is for

- MacBook / iMac / Mac mini / Mac Studio running **macOS 12+**
- Intel Macs: PyTorch runs on CPU; training will be slower
- Apple Silicon (M1/M2/M3/M4): use **MPS** backend for GPU-like acceleration
