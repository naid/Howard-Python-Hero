Complete these three modules **in order** before starting shared lessons in [`../lessons/`](../lessons/).

| Order | Module | Goal |

| 1 | [01-system-setup](01-system-setup.md) | Terminal, packages, build tools |

## Who this track is for

- **Debian 12 (Bookworm)** or **Ubuntu 22.04+** (commands are nearly identical)
- Desktop or server with SSH access
- NVIDIA GPU recommended for deep learning (CPU-only is possible but slow)
