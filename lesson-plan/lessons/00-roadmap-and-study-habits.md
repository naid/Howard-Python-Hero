## What you are preparing for

The Tesla AI intern role focuses on:

- **Large-scale models** for autonomy (self-driving)
- **Data engines** — curating massive driving datasets
- **Foundation models** — vision, language, generative video
- **Reinforcement learning** — robustness and training objectives

You are not expected to master everything in one month. This course builds layer by layer.

## Glossary (learn gradually)

| Term | Plain English |
|------|----------------|
| **Model** | A mathematical function learned from data |
| **Training** | Adjusting model parameters to reduce error |
| **Inference** | Using a trained model on new data |
| **Loss** | Number measuring how wrong predictions are |
| **Gradient** | Direction to adjust weights to reduce loss |
| **Epoch** | One pass through the training dataset |
| **Batch** | Subset of data processed together |
| **CNN** | Neural net for images |
| **Transformer** | Architecture behind GPT and many modern models |
| **RL** | Learning from rewards/actions, not just labels |
| **World model** | Model that predicts how the environment evolves |

## How to study effectively

1. **Code every day** — 30 minutes beats 5 hours once a week.
2. **Struggle, then look** — Try exercises 15 minutes before reading solutions.
3. **Explain aloud** — Teach the concept to an imaginary colleague.
4. **One project per phase** — Folder under `~/python-hero/projects/`.
5. **Read papers late** — Lessons 9+ link papers; skim abstracts first.
